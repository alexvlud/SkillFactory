Итоговое задание по ООП. Задание 23.6.1
--------------------------------
Телеграм-бот для конвертации валют. 
Имя в телеграме: @al_currency
Файлы:
    - app.py - основной функционал
    - extensions.py - классы 
    - config.py - Токен telegramm-бота и список валют
Необходимые пакеты: 
    - requests
    - PyTelegramBotAPI
    - json

Для запуска бота загрузите app.py, extensions.py и config.py на свой компьютер и запустите app.py.
